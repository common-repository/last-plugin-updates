Shows a list of installed plugins and the date of their last update.

== Description ==

This plugin adds a widget to the dashboard, listing the installed plugins and the date of their last update. This list can be displayed in ascending and descending order of name and date of the last update.

== Installation ==

1. Upload all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy the plugin on the Dashboard
